Pertanyaan : Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach?

Jawaban :

  • IF ELSE sering saya gunakan untuk melakukan percabangan. Jadi jika suatu kondisi benar, maka program akan mengeksekusi kode tertentu tetapi jika tidak maka program akan mengeksekusi kode yang lainnya
  • WHILE sering saya gunakan untuk melakukan perulangan selama suatu kondisi tersebut masih true/benar. Kondisi akan terus di cek setiap kali perulangan berlangsung, dan perulangan akan berhenti jika kondisi sudah false
  • DO WHILE merupakan perulangan yang mirip dengan while, namun do while akan melakukan perulangan setidaknya satu kali
  • Switch case hampir mirip dengan if else, jadi sering saya gunakan untuk melakukan percabangan, tetapi berdasarkan nilai suatu variabel (ekspresi). Jika nilai variabel atau ekspresi cocok dengan yang didefinisikan, maka program akan mengeksekusi kode tertentu
  • For sering saya gunakan untuk melakukan perulangan dengan jumlah iterasi yang sudah diketahui sebelumnya
  • Foreach sering saya gunakan untuk melakukan perulangan pada setiap elemen dalam suatu array atau objek

Pertanyaan : Apa itu constructor, method, class, object, variabel, parameter, function, procedure ?

Jawaban :

  • Constructor merupakan sebuah fungsi khusus yang akan dieksekusi oleh program pertama kali atau saat objek dibuat, constructor ini sering digunakan untuk menginisialisasi variabel dan melakukan tugas-tugas awal lainnya
  • Method adalah fungsi yang didefinisikan dalam suatu kelas. Method digunakan untuk melakukan tugas-tugas tertentu pada objek kelas tersebut
  • Class adalah sebuah blueprint atau cetakan untuk membuat objek. Class mendefinisikan atribut dan method yang dimiliki oleh objek
  • Object adalah sebuah instance dari suatu kelas. Objek memiliki atribut dan method yang ditentukan oleh kelasnya
  • Variabel adalah suatu nilai yang disimpan dalam memori dan dapat diubah selama program berjalan
  • Parameter adalah nilai yang diteruskan ke dalam suatu fungsi atau method. Parameter dapat digunakan dalam fungsi atau method untuk melakukan tugas tertentu
  • Function adalah blok kode yang diberi nama dan dapat dipanggil dari bagian lain dalam program. Fungsi dapat mengembalikan nilai atau tidak
  • Procedure adalah blok kode yang diberi nama dan dapat dipanggil dari bagian lain dalam program. Prosedur tidak mengembalikan nilai

Pertanyaan : Perbedaan method="Post" dan "Get" ?

Jawaban : Post dan Get dalam sebuah form sama sama digunakan untuk mengirimkan data dalam http request, tetapi jika Data yang dikirim melalui "GET" itu ditambahkan ke URL sebagai parameter query string, sedangkan data yang dikirim melalui "POST" dikirimkan dalam body permintaan HTTP. kemudian kapasitas data yang dapat dikirimkan melalui "GET" itu terbatas, sedangkan "POST" dapat lebih besar lagi, dan tentunya method post lebih aman karena data yang dikirimkan tidak terlihat oleh user

Pertanyaan : Jelaskan konsep mengenai web service dan fungsi apa yang digunakan untuk memanggil web service client ?

Jawaban : Web service adalah teknologi yang memungkinkan dua atau lebih aplikasi berbeda untuk berkomunikasi melalui jaringan dengan menggunakan protokol HTTP. Web service menggunakan format data XML atau JSON untuk bertukar data antar aplikasi. Untuk memanggil web service client, kita dapat menggunakan fungsi "curl" pada PHP atau menggunakan perpustakaan atau framework khusus yang menyediakan fungsi untuk memanggil web service. Fungsi "curl" dapat digunakan untuk mengirim permintaan HTTP ke server dan menerima respons HTTP.

Pertanyaan : Jelaskan mengenai MVC dan HMVC !

Jawaban : MVC (Model-View-Controller) adalah pola desain arsitektur perangkat lunak yang memisahkan aplikasi menjadi tiga bagian utama: model (yang merepresentasikan data dan logika bisnis), view (yang menampilkan data ke pengguna), dan controller (yang mengatur alur aplikasi). Pola MVC memungkinkan aplikasi untuk dibangun secara terstruktur, mempermudah perawatan, dan memungkinkan pengembangan bersama oleh tim. Sedangkan HMVC (Hierarchical Model-View-Controller) adalah pengembangan dari pola MVC yang memungkinkan pengembangan aplikasi menjadi sub-modul yang lebih kecil dan lebih terisolasi. HMVC menggunakan pola MVC dalam setiap sub-modul, yang memungkinkan aplikasi untuk memperoleh fleksibilitas yang lebih besar.

Jawaban : Perbedaan antara HTTP dan HTTPS adalah tingkat keamanan. HTTPS menggunakan enkripsi untuk melindungi data yang dikirim, sedangkan HTTP tidak. Selain itu, HTTPS menggunakan port yang berbeda (443) daripada HTTP (80).
